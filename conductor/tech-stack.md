# Technology Stack

## Programming Languages

- **Python:** Used for the backend API and data processing pipelines.
- **TypeScript:** Used for the Angular frontend application.
- **HCL (HashiCorp Configuration Language):** Used for Terraform infrastructure definitions.

## Frameworks & Libraries

- **FastAPI:** High-performance web framework for the Python backend.
- **Angular:** Frontend framework for building the web application.
- **Pydantic:** Data validation and settings management for the backend.
- **Uvicorn:** ASGI server for running the FastAPI application.

## Cloud & Infrastructure

- **Google Cloud Platform (GCP):** Primary cloud provider.
- **Gemini:** Google's AI model for video analysis and product identification.
- **BigQuery:** Data warehouse for storing product data and analysis results.
- **Google Merchant Center:** Source of truth for retailer product inventory.
- **Cloud Run:** Likely hosting for the backend and frontend (based on `cloud_run` directories).
- **Terraform:** Infrastructure as Code (IaC) for managing GCP resources.

## Data & Storage

- **Google Cloud Storage (GCS):** Storage for video files.
- **Vector Search:** Used for semantic product matching via embeddings.

## Tooling

- **Git:** Version control.
- **Pyink / Black:** Python code formatting (Google style).
- **GTS (Google TypeScript Style):** TypeScript linting and formatting.
- **Pip-compile:** Managing Python dependencies.
